# Neovim Cheatsheet

## 1. ファイルツリーの表示/非表示

| 操作 | キー |
|------|------|
| トグル表示 | `<Space>e` |

ツリー内では:
- `Enter` — ファイルを開く
- `a` — 新規ファイル/ディレクトリ作成
- `d` — 削除
- `r` — リネーム
- `?` — ヘルプ表示

## 2. ペイン分割

| 操作 | キー (推奨) | コマンド |
|------|------|------|
| 横分割 (上下) | `<C-w>s` | `:split` / `:sp` |
| 縦分割 (左右) | `<C-w>v` | `:vsplit` / `:vsp` |

例えば `:vsp ~/.zshrc` で右に .zshrc を開く、もできる。

## 3. ペイン削除

| 操作 | キー |
|------|------|
| 現在のペインを閉じる | `:q` |
| 他のペインを全閉じ (今のだけ残す) | `<C-w>o` |

## 4. ペイン間移動

| 操作 | キー |
|------|------|
| 左に移動 | `<C-h>` |
| 下に移動 | `<C-j>` |
| 上に移動 | `<C-k>` |
| 右に移動 | `<C-l>` |

`<C-w>h/j/k/l` でも同じ動作 (vim 標準) だが、設定済みのショートカットだと `<C-w>` を打たずに済む。

## 練習シナリオ

```text
1. nvim を起動
2. <Space>e        → 左にファイルツリー表示
3. ツリーでファイルを Enter で開く
4. <C-w>v          → 右に縦分割
5. 右側で :e ~/.zshrc など別ファイルを開く
6. <C-h> / <C-l>   → ペイン間を移動
7. :q              → 今いるペインを閉じる
8. <Space>e        → ツリーを閉じる
```

## ファジーファインダー (Telescope)

| 操作 | キー |
|------|------|
| ファイル検索 | `<Space>f` |
| プロジェクト内 grep | `<Space>g` |
| 開いてるバッファ一覧 | `<Space>b` |
